## 基础指令
```text
# /SetHome [Name]
- 设置一个家 (若不填写Name默认设置为"Home")
# /Home [Name]
- 返回家 (若不填写Name默认返回"Home")
# /DelHome [Name]
- 删除家 (若不填写Name默认删除"Home")
# /HomeList
- 列出所有家的位置
# /Tpa <玩家>
- 发送一个传送请求给玩家
# /Tpaccept
- 接受一个玩家的传送请求
# /Tpadeny
- 拒绝一个玩家的传送请求
# /TpaHere <玩家>
- 请求一个玩家传送到身边
# /Friends List
- 查看好友列表
# /Friends add <玩家>
- 发送好友请求给一个玩家
# /Friends deny <玩家>
- 拒绝玩家的好友请求
# /Friends tp <玩家>
- 传送至一个好友
# /Friends tpstatus (true | false)
- 设置好友之前的传送是否开启
# /Friends status (message | reset)
- 设置自己当前的状态
# /Friends remove <玩家>
- 删除一个好友
```
